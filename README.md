# Sistema Integral de Gestión de Clientes, Servicios y Reservas

## Descripción

Este proyecto fue desarrollado en Python utilizando programación orientada a objetos para la empresa ficticia Software FJ.

El sistema permite gestionar clientes, servicios y reservas sin utilizar bases de datos, implementando listas, objetos y manejo de archivos.

Además, el programa incorpora manejo avanzado de excepciones para garantizar estabilidad y robustez durante la ejecución.

---

## Funcionalidades

- Registro de clientes
- Validación de datos
- Gestión de servicios
- Creación de reservas
- Cancelación de reservas
- Manejo de errores
- Registro de logs
- Programación orientada a objetos

---

## Tecnologías utilizadas

- Python
- Programación Orientada a Objetos
- Manejo de excepciones
- Archivos de logs
